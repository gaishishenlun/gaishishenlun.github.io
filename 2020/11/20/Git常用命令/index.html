Git常用命令

Git常用命令

官方文档

man git

修改git身份

git config --global user.name "XXX"
git config --golbal user.email "XXX@XX.com"

仓库初始化

git init

查看仓库状态

git status

提交(添加)仓库修改

git add

查看未提交的修改，比较工作区与暂存区的区别

git diff

撤销更改(撤销add操作)

git reset	#全部撤销
git restore	#撤销某个文件

提交内容(版本)

git commit -m "XXX"	#XXX随意，一般用来版本描述

取消追踪

git rm --cached XX	#XX为文件名
编辑	.gitignore	 添加文件即可不追踪，已追踪的文件需要先取消追踪。

分支操作

git branch XX		#添加分支
git checkout XX		#切换分支
git merge XX		#合并分支
git brancjh			#列出本地分支
git	branch -d XX	#删除分支	-D为强制删除

远程仓库

git remote add origin XXX	(XXX为远程地址 例：github.com/XXX.git)		#添加远程仓库
git push --set-upstream		#设置本地分支追踪远程分支	
git clone XXX				#克隆仓库
